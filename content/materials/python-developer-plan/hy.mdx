## Ծրագրի նպատակ

Այս ուղին բազային մակարդակի սովորողներին հասցնում է գործնական Python մշակողի մակարդակի։ Յուրաքանչյուր մոդուլ ավարտվում է կիրառական առաջադրանքներով, որոնք մոտ են իրական մշակման միջավայրին։

## Իրականացման մոդել

- Տևողություն: 16 շաբաթ
- Հաճախականություն: շաբաթական 2 դաս
- Դասի տևողություն: 90 րոպե
- Ընդհանուր դասերի քանակ: 32

## Մոդուլային բաժանում

### Փուլ 1: Python հիմունքներ (1-4 շաբաթ)

- Python շարահյուսություն և մաքուր կոդի սկզբունքներ
- Կառավարման հոսք՝ պայմաններ և ցիկլեր
- Տվյալների կառուցվածքներ՝ list, dictionary, tuple
- Ֆունկցիաներ և մոդուլային կառուցվածք

### Փուլ 2: Խնդիրների լուծում և տվյալներ (5-8 շաբաթ)

- Տողերի և ցանկերի խնդիրներ
- Ֆայլային մուտք/ելք և CSV հիմունքներ
- Սխալների մշակում `try/except`-ով
- Փոքր նախագիծ 1. Տվյալների մշակման CLI գործիք

### Փուլ 3: Գործնական ավտոմատացում և API (9-12 շաբաթ)

- Մոդուլներ, փաթեթների օգտագործում և վերաօգտագործվող կոդ
- HTTP հարցումներ և JSON պատասխանների մշակում
- Տեղային ավտոմատացման սցենարներ
- Փոքր նախագիծ 2. API-ով աշխատող հաշվետվության գործիք

### Փուլ 4: Պրոդուկտի թողարկման ցիկլ (13-16 շաբաթ)

- Պրոդուկտի նպատակի և կառուցվածքի սահմանում
- Իրականացման սպրինտ
- Ռեֆակտորինգ, թեստավորում, փաստաթղթավորում
- Եզրափակիչ պաշտպանություն և հանրային ցուցադրում

## Գնահատման համակարգ

- Շաբաթական լաբորատոր և հանձնարարություններ: 40%
- Ստուգումներ և թեմատիկ հարցումներ: 20%
- Փոքր նախագծեր և եզրափակիչ պրոդուկտ: 40%

## Եզրափակիչ նախագծի պահանջներ

- Հստակ սահմանված խնդիր և ծավալ
- Աշխատող Python հավելված
- Թեստային դեպքեր և դեբագի ապացույցներ
- Կարճ տեխնիկական փաստաթուղթ
- Կենդանի դեմո և հարցուպատասխան

## Շրջանավարտի պրոֆիլ

Այս ուղին ավարտած ուսանողները պատրաստ են առաջադեմ ուղղությունների՝ backend մշակման, ռոբոտաշինական ծրագրավորման կամ տվյալների վրա հիմնված նախագծերի համար։
